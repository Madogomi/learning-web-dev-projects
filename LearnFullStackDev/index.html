Learning to code

I will upload videos and playlists that i watched while learning full stack website development.

This is a step by step guide.

But, DO NOT JUST WATCH THESE VIDEOS, actually go out there and build, play around with code, break things, fix them, grow, google for how to do stuff that you don't know or understand. We even have stuff like ChatGPT, Claude AI and Gemini and other AIs to help and assist you on your learning journey.

But, BEWARE: DO NOT LET CHATGPT AND ANY OTHER AI BECOME A CRUTCH YOU CAN'T LIVE WITHOUT, WRITE THE CODE YOURSELF, DON'T JUST COPY THE CODE AND SAY IT'S FINE BECAUSE YOU UNDERSTAND THE CODE, WRITE THE CODE YOURSELF, PLAY AROUND WITH IT, CHECK OUT WHAT WORKS AND WHAT DOESN'T WORK.

These are the front-end videos/playlists

This is the playlist for HTML(Hyper Text Markup Language) and CSS(Cascading Style Sheets)

HTML and CSS should be the first thing you learn on your path to web development. It is very important. It is the foundation of website development. Don't listen to bad advice like 'Learn React' or 'Learn Angular'. HTML, CSS and JS are the basics of web development.

This is a boiler plate for beginners

This contains everything your website will actually need, you can add more to it, but this is essential.

If you don't know what a boilerplate is, just google it, or ask any AI.


                <!DOCTYPE html>
                <!-- Declares this document as an HTML5 file -->

                <html lang="en">
                  <head>
                    <meta charset="UTF-8" />
                    <!-- Sets character encoding to UTF-8 for full character support -->

                    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                    <!-- Ensures your website is responsive on all devices so that it doesn't look ugly and bad on devices with smaller screens -->

                    <title>Your Website Title</title>
                    <!-- This is the title that shows in the browser tab and when they search for your website -->

                    <link rel="stylesheet" href="styles.css" />
                    <!-- Links your external CSS file (you can rename it, it doesn't have to be styles.css, it could be random.css.
                     Just make sure there is a file called random.css) -->
                  </head>

                  <body>
                    <header>
                      <!-- This is the top section of your site: logo, nav bar, etc. -->
                    </header>

                    <main>
                      <!-- Main content of your website goes here -->
                    </main>

                    <footer>
                      <!-- Bottom section of your site: copyright, links, etc. -->
                    </footer>

                    <script src="script.js"></script>
                    <!-- Links your external JavaScript file (You can rename it just like the styles.css file,
                    you could name it site.js as far as you have a website called site.js) -->
                  </body>
                </html>

These are the playlist/videos for Git and GitHub

These are the commands to help you with git and GitHub

You can watch youtube videos to learn more about this, but the tutorial should teach you more than enough

git config --global user.name "Your-Name"

That connects you to your GitHub account so that GitHub may know who you are

git config --global user.email "your@email.com"

This connects you to the email you used to open the GitHub account

git init

This initializes the git in your project folder

git status

This tells you the status of your project

git add .

This saves everything and all the changes you have made in your folder to the git repository

git add filename

This is if you only want to save a specific file to the git repository

git commit -m "Your message here"

This will now commit your saved changes. Please always use clear messages so that other people or future you can come back and know what actually happened

git remote add origin https://github.com/yourusername/yourrepo.git

This will connect you to your GitHub account

git push -u origin main

This will push your changes to your main branch, you can change main to the name of any other branch

git push

This will just push to the branch you're currently on

git pull

This is mainly incase you're working with someone and you want to see and get the latest changes they have made to the main branch

git log

This will show all your commit history

git clone https://github.com/username/repo.git

This is mainly to download someone else's project

git restore filename

This is to revert a certain file back to the last commit incase you made a mistake or messed up

git reset --hard HEAD

This is to revert the whole project back to the last commit incase you made a big mistake or messed up badly
